L'ASBL est le Royal Cercle de Voile de Dave (RCVD).
- Site web : https://www.royal-cercle-de-voile-de-dave.be/
- Localisation : Dave, vallée de la Meuse, Belgique
- ~300 membres, 15 bateaux, ~90 élèves formés/an
- Activités : école de voile (jeunes + adultes), régates, événements sociaux
- Clubhouse : une péniche en location à ~1€/an par la commune (info à vérifier)
- Petite ASBL, non assujettie à la TVA
- Comptabilité simplifiée

**Why:** Le projet OpenASBL est d'abord construit pour répondre aux besoins réels de cette ASBL avant d'être généralisé.
**How to apply:** Utiliser ce cas concret pour guider les choix de design (catégories par défaut, types de recettes/dépenses typiques d'un club sportif).
