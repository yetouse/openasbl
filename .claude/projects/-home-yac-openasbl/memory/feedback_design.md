La couleur primaire doit être bleu marine (#1e3a5f), pas le bleu Bootstrap par défaut qui est trop agressif.

**Why:** Yannick trouve le bleu vif de Bootstrap trop agressif pour une app comptable.
**How to apply:** Toujours utiliser le bleu marine défini dans style.css pour les éléments primaires (navbar, boutons). Privilégier un design sobre et professionnel.
